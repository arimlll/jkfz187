聆听《如何让小野猫爱上我？》

  • 步骤一

    • 点按 复制 RSS 订阅链接 。

  • 步骤二

    • 使用 iOS

    1. 打开“播客”App。
    2. 轻点“资料库”。
    3. 轻点右上角的“编辑”，然后选择“通过 URL 关注节目...”。
    4. 输入 RSS 订阅链接。
    5. 轻点“关注”。

    使用 MacOS

    1. 打开“播客”App。
    2. 点按“文件”，然后选择“通过 URL 关注节目...”。
    3. 输入 RSS 订阅链接。
    4. 选择“关注”。

  • 步骤三

    • 尽情聆听本播客。
